- HTML5 no es solo la especificación de marcado HTML5, sino que la familia HTML5 incluye también CSS3, JS y otros elementos:
	- Geolocalización
	- Almacenamiento web
- La web tiene mayor capacidad y velocidad, lo que permite mejorar la experiencia web y acercarla a las aplicaciones de ordenador:
	- Editor de imágenes
	- Mapas
	- Office online
	- Multimedia
- XHTML y XML no eran adecuados. Se trabajó en XHTML 2 pero se abandonó. Se desarrolló HTML5 a partir de HTML4  (originariamente: era Web Applications 1.0).
- HTML5 aporta:
	- Estructura más lógica y funcional (del div a cada elemento propio)
	- Elementos nativos multimedia: la muerte de los plugins para reproducir vídeo y audio
	- Semántica con los nuevos elementos y metadata => SEO
	- Geolocalización para enriquecer las aplicaciones => Mapas
	- Web Storage para guardar variables en el ordenador del usuario aporta más potencia que los mecanismos anteriores (cookies): mayor cantidad de datos, más fácil y eficiente. 2 opciones en HTML5: localStorage (permanecen al cerrar el navegador) y sessionStorage (se pierden al cerrar el navegador)
